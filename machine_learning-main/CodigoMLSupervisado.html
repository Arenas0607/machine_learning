Caso de éxito en Machine Learning Supervisado

¿Qué es Machine Learning?

El 'machine learning' o aprendizaje automático es una rama de la inteligencia artificial que permite que las máquinas aprendan ciertas tareas de algoritmos, sin que necesariamente estén programadas de manera específica para ello.

Es básicamente, el reconocimiento de patrones, predicciones y recomendaciones a partir del procesamiento de grandes datos, todo para lograr las mejores decisiones y proyecciones en base a ese análisis.

El Machine Learning es una rama de la Inteligencia Artificial.

En aplicaciones prácticas lo vemos en las sugerencias personalizadas de compra en plataformas online o recomendaciones de canciones. En su forma más básica analiza el historial de compras y reproducciones del usuario y lo compara con lo que han hecho otros usuarios con tendencias o gastos parecidos.

Spotify, Youtube y las principales plataformas de streaming lo usan para recomendar nuevos contenidos que mantengan al usuario en la página durante más tiempo. La red social X, por ejemplo, se sirve de algoritmos de machine learning para reducir en gran medida el spam publicado.

¿Qué es Machine Learning Supervisado?

El aprendizaje supervisado es el uso de conjuntos de datos etiquetados para entrenar algoritmos que clasifican los datos o predicen los resultados con precisión. Este tipo de machine learning debe su nombre a que la máquina está "supervisada" mientras aprende, lo que significa que se alimentan a los algoritmos con información para ayudarles a aprender.

El resultado que se le proporciona a la máquina son datos etiquetados, y el resto de la información que se le da se utiliza como características de entrada.

Entre los algoritmos más utilizados en el aprendizaje supervisado se encuentran las redes neuronales (neural networks) y los árboles de decisión (decision trees).

Caso de éxito

En la Universidad Politécnica de Madrid, para un proyecto de grado en Ingeniería de Software, el ingenierio Juan Sebastian Moreno y su director Fernando Ortega, desarrollaron modelos de ML con algoritmos de aprendizaje supervisado para predecir ataques cardíacos. "Utilizaremos una base de datos que contiene datos médicos de varios pacientes, y compararemos diferentes algoritmos de aprendizaje automático para averiguar con cuál podemos obtener mejores resultados prediciendo si se producirá un infarto o no."

Se probaron diferentes algoritmos de aprendizaje supervisado y técnicas de preprocesamiento para determinar cuál ofrece mejores resultados en términos de precisión y capacidad de predicción. Se compararon métodos de sobremuestreo (oversampling) y submuestreo (undersampling) para equilibrar las clases en el dataset. Finalmente, se concluyó que los algoritmos más efectivos fueron Logistic Regression y Random Forest Classifier, con un F1-score del 85% y solo tres falsos negativos.

Prevenir ataques cardíacos mediante ML.

La conclusión indica que los algoritmos Logistic Regression y Random Forest Classifier fueron los más efectivos en la predicción de ataques cardíacos, alcanzando un F1-score del 85%, lo que significa que lograron un buen equilibrio entre precisión (predicciones correctas de ataques) y sensibilidad (capacidad de detectar los casos reales). Además, solo generaron tres falsos negativos, es decir, solo en tres casos el modelo no detectó un ataque cardíaco cuando realmente ocurrió. Esto es clave en el ámbito médico, ya que minimizar los falsos negativos es fundamental para evitar que pacientes en riesgo no reciban atención oportuna.

La alta precisión del modelo mejora el diagnóstico y reduce la dependencia exclusiva de la evaluación médica tradicional, ayudando a los profesionales de la salud a tomar mejores decisiones basadas en datos. Esto no solo optimiza los recursos médicos, sino que también contribuye a salvar vidas al proporcionar alertas oportunas y mejorar los tratamientos preventivos.

Ejemplos de empresas usando Machine Learning

Netflix: Usa Machine Learning supervisado para predecir qué series o películas pueden interesar a los usuarios. El modelo se entrena con datos etiquetados sobre lo que los usuarios han visto, puntuaciones y tiempo de visualización para recomendar contenido similar.

Pay Pal: Usa Machine Learning supervisado para identificar transacciones fraudulentas. El modelo se entrena con datos de transacciones etiquetadas como "fraudulentas" o "legítimas" para detectar patrones y bloquear actividades sospechosas en tiempo real.

Tesla conducción automática: Tesla entrena su sistema de Autopilot con Machine Learning supervisado, usando millones de imágenes y videos etiquetados de situaciones de tráfico reales. El modelo aprende a identificar peatones, señales de tránsito y otros vehículos para mejorar la conducción autónoma.
